# SKILL 06 — Economia Virtual & Progressão

## Filosofia Econômica
O jogo deve ser **completável gratuitamente** com esforço consistente.
Pagamentos aceleram a jornada mas nunca são obrigatórios. A moeda virtual
(Fúria Coins) deve manter valor percebido sem inflacionar.

## 2. Balanceamento Econômico

### Regra de Sustentabilidade
```
TAXA DE ENTRADA/SAÍDA ESPERADA (usuário médio):
Entrada diária: ~85 coins (login + 1 missão + 1 troca)
Entrada semanal: ~700 coins
Custo de 1 pacote básico/dia: 100 coins
→ Gap de ~100 coins/semana → usuário cria "poupança" gradual
→ A cada ~7 dias pode comprar 1 pacote extra sem esforço adicional
```

### Proteções Anti-Inflação
- Máximo de 10.000 coins no "bolso" (excedente vai para "cofre" sem juros)
- Conversão de repetidas: taxa diminui conforme mais cópias (5ª cópia = 1 coin)
- Figurinhas lendárias: máximo de 3 conversões por lendária por mês

### Regras de Monetização Ética
- **Nunca** vender figurinhas específicas diretamente (apenas pacotes aleatórios)
- **Nunca** criar vantagem competitiva absoluta via pagamento (balance é sobre velocidade, não acesso)
- Limite de gasto: R$ 200/mês por conta (proteção de menores e adultos vulneráveis)
- Aviso claro de "este é um jogo virtual, figurinhas não têm valor real"
- Reembolso disponível em até 24h para compras acidentais

## 5. Integração Phygital — Código Fúria

### Funcionamento
```
FLUXO PHYGITAL:
1. Usuário compra pacote físico (versão física futura)
2. Dentro do pacote: cartão com QR Code / "Código Fúria" de 12 dígitos
3. No app: menu "Resgatar Código Fúria"
4. Código válido: 1 pacote digital Elite bônus + 1 figurinha exclusiva "Edição Física"
5. Código inválido/já usado: mensagem de erro clara
6. Cada código é de uso único e vinculado ao usuário que resgatou
```

### Figurinhas Exclusivas Phygital
- Série "Edição Física" — só obtenível via Código Fúria
- Borda diferenciada (efeito papel/textura)
- Indicador "🎴 Edição Física" no card
- Não podem ser trocadas (são pessoais, da experiência phygital)

## Regras Imutáveis desta Skill
- Fúria Coins nunca vencem ou expiram
- Preços em IAP só podem mudar com 30 dias de aviso antecipado
- A taxa de ganho free-to-play jamais pode ser reduzida para forçar compras
- Toda feature de monetização deve ter equivalente gratuito (mais lento)
- Relatório mensal de economia deve ser monitorado para detectar desequilíbrios
